# Kraken

> The backed service for the Seantinel Project built with NodeJS

### Main Features:

- Supports Real-time Updates via Websockets using Socket.io
- Supports Rest API endpoints
- Open, Cross Origin Request Enabled.

### Getting Started:

**with yarn:**

Install Dependencies

```sh
yarn
```

Development:

```sh
yarn start
```

Build:

```
 yarn build
```

**with npm:**

Installation:

```sh
npm i
```

Development:

```sh
npm start
```

Build:

```
 npm build
```
